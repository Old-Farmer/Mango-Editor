# Keymaps

A keymap is a key sequence which can trigger some operations. Sometime keymaps are called Shortcuts in other editors.

NOTE: Don't support self-defined keymaps now.

`<c-...>` means ctrl + ...

`<a-...>` means alt + ...

Keymaps is very similar in vim/neovim, differences:

1. Arrow keys, home/end, pgup/pgdn key is not set.
2. When Peel(the area below the status line) is expand to multi-row, you can use `<enter>` to go into Peel and navigate in it.
3. Completion is triggered by `<c-space>` or `<c-c>`
