# Commands

Commands are instructions you can execute in the Mango Peel. You can use full form or short form as you like.

- `quit`  
    short form: q
    desc: quit the editor

- `quit!`  
    short form: q!
    desc: force quit the editor

- `write [path]`  
    short form: w
    desc: save the current content as a file, or save as another file if `path` is specified.

- `help [doc-name]`  
    short form: h
    desc: goto help

- `edit <path>`  
    short form: e
    desc: goto and edit the file

- `buffer <path>`  
    short form: b
    desc: goto and edit the buffer

- `bdelete`
    short form: bd
    desc: delete the current buffer

- `smile`
    short form: None
    desc: print a smile.
