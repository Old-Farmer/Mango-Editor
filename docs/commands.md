# Commands

Commands are instructions you can execute in the Mango Peel. You can use full form or short form as you like.

- `search <search-pattern>`  
    short form: s
    desc: search string

- `saveas <path>`  
    short form: sa
    desc: save the current content as a file

- `help [doc-name]`  
    short form: h
    desc: goto help

- `edit <path>`  
    short form: e
    desc: goto and edit the file

- `buffer <path>`  
    short form: b
    desc: goto and edit the buffer

- `goline <line>`
    short form: g
    desc: goto the line.

- `smile`
    short form: None
    desc: print a smile.
